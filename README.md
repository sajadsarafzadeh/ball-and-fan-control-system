# Ball and Fan Control System

This project presents the design and implementation of a **Ball and Fan control system** developed to stabilize the position of a lightweight ball using airflow control. The project combines **system modeling, control design, and experimental validation**.

## Overview

The purpose of this project is to control the position of a ball in a vertical airflow system by adjusting the fan input. The setup demonstrates a classic **feedback control problem** where the system must continuously react to disturbances and maintain stability.

## Objectives

- Model the dynamics of the ball-and-fan system
- Design a suitable controller for stabilization
- Analyze system behavior and response
- Validate the control performance experimentally or through simulation

## System Components

- Fan / actuator
- Ball
- Position sensing mechanism
- Control algorithm
- Data acquisition / processing unit

## Control Method

A feedback controller was implemented to regulate the ball position. The controller was tuned based on system response in order to improve:

- Stability
- Rise time
- Overshoot
- Steady-state error

## Results

The final system was able to maintain the ball around the desired position with acceptable dynamic response. Performance evaluation was carried out using response plots and/or experimental observations.
